help files are made with the help 2 0 workshop vshik a help authoring tool the default viewer for help 2 x files is microsoft document explorer and there are several third party viewers available such as h2viewer and help explorer viewer visual studio 2010 uses a new help engine microsoft help viewer a microsoft help 2 x file has a hxs extension a compressed hxs help file help title is compiled from a set of topic pages written in a subset of html a hxc main project file an hxf include file a hxt table of contents a hxa attribute definition file and a number of hxk indexes keyword index namedurl index optional associated and context links indexes this format was originally intended only for the help system used by visual studio net 2002 help msdn library and technet but now it is used in office 2007 and some ides such as borland developer studio
