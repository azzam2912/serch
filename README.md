## Installation
1. Clone the repository
2. Install the requirements
3. Run the server
4. Open the web app at [localhost:5000](http://localhost:5000)

```bash
git clone https://github.com/azzam2912/serch.git
cd serch
pip install -r requirements.txt
python app.py
```
